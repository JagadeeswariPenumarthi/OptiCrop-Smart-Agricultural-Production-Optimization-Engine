About OptiCrop

OptiCrop is a machine learning powered web application that recommends the most suitable crop to grow based on soil nutrient levels and climate conditions. It is intended as a decision-support tool for farmers and agricultural planners.

Objectives

  • Analyze soil and climate data to identify patterns linked to crop suitability.
  • Train a classification model capable of recommending a crop from seven input features.
  • Provide a simple, accessible web interface for real-time predictions.
  • Support more informed, data-driven crop selection to improve yield and sustainability.

Methodology

  1. Load and inspect the crop recommendation dataset (2,200+ records).
  2. Clean the data: check for missing values and duplicate rows.
  3. Detect and remove outliers using the Interquartile Range (IQR) method.
  4. Explore the data with boxplots and simple seasonal crop groupings.
  5. Split the data into training (80%) and testing (20%) sets.
  6. Train a Logistic Regression classifier (with feature scaling) to predict the crop label.
  7. Use K-Means clustering and the Elbow Method as a complementary unsupervised analysis.
  8. Evaluate the model using accuracy, precision, recall, F1-score, and a full classification report.
  9. Save the trained model to model.pkl for use by the Flask application.

Inputs Used for Prediction

Nitrogen (N)
Phosphorous (P)
Potassium (K)
Temperature (°C)
Humidity (%)
Soil pH
Rainfall (mm)
